Bio profesional

Soy Técnico en Redes y Soporte certificado por el Instituto de Capacitación Técnica KEYS en Caracas, Venezuela. Desde 2006, he acumulado una amplia experiencia en informática, desempeñándome como capacitador en estructura de computadoras, analista de soporte, técnico en instalación y mantenimiento, y administrador de redes de datos y sistemas CCTV.

Como programador autodidacta me especializo en JavaScript, typescript y PHP. Trabajando tanto de manera freelance como a nivel empresarial he desarrollado habilidades en diseño, maquetación y desarrollo de sitios web que me han permitido crear soluciones personalizadas y optimizadas utilizando patrones de diseño y paradigmas como la programación estructurada y orientada a objetos (POO).

Comprometido con las mejores prácticas, organizo mi código siguiendo principios como SOLID, la filosofía DRY y el uso de patrones de diseño. Esto me permite minimizar la deuda técnica y evitar refactorizaciones innecesarias, incluso cuando trabajo bajo tiempos de entrega ajustados.

He liderado proyectos desde cero, gestionando todas las etapas: levantamiento de requerimientos, diseño, desarrollo, pruebas, implementación, monitoreo y actualización de aplicaciones. Trabajo bajo la metodología SCRUM, fomentando una comunicación efectiva y un ambiente de respeto para alcanzar resultados eficientes y de alta calidad.

Mis proyectos incluyen soluciones desarrolladas con JavaScript, PHP, y Node.js con Express, diseñando interfaces de usuario modernas, intuitivas y adaptables, integradas con bases de datos como SQLite, MySQL, SQL Server (en colaboración con analistas de datos para minería basada en modelos de negocios) y MongoDB en entornos prácticos y personales.

Términos y condiciones de servicios

Introducción

Este acuerdo establece los términos y condiciones bajo los cuales yo Railer Chalbaud (en adelante, "el Proveedor") proporcionare servicios de desarrollo web full stack a Ud. como Cliente.

Alcance de los Servicios

El Proveedor se compromete a desarrollar y entregar un proyecto web personalizado al Cliente, según las especificaciones acordadas en el contrato.

Tarifas

  • Costo mínimo de desarrollo: $15 USD/hora
  • Costo mínimo por proyecto: $350 USD.
  • Tiempo de entrega: Un mínimo de 7 días, sujeto a la complejidad del proyecto.
  • Cambios posteriores a la entrega: Se incluyen hasta 3 cambios sencillos sin costo adicional.
  • Costo de actualizaciones: Todas las actualizaciones bien sean estética, de contenido o funcionales se cobran por hora.
    1. Funcionalidad:Actualizaciones que incluyan una nueva funcionalidad a partir de $45 (3hr).
    2. Estética y contenido: Actualizaciones en imágenes y contenido textual a partir de $15 (1hr).
  • Servicio de mantenimiento: Este es opcional y tiene un costo inicial de $30 USD mensuales con un límite de 15 cambios por mes que solo incluyen cambios estéticos y de contenido.
  • Pagos: El cliente puede realizar sus pagos en Bs. A través de pago móvil, divisas en efectivo (previo acuerdo) o divisas digitales mediante Zinli.

Proceso de Trabajo

  • Comunicación: Toda comunicación entre el Cliente y el Proveedor se realizará principalmente por WhatsApp, Telegram o correo electrónico, las reuniones presenciales serán solo si la ubicación lo permite y si es estrictamente necesario, previo acuerdo.
  • Requisitos: El Cliente es responsable de proporcionar toda la información necesaria para el desarrollo del proyecto, incluyendo conexiones a BD (en caso de ser necesario), contenido textual, imágenes y cualquier otro material relevante. Es vital que el cliente tenga una idea clara de lo que quiere, para evitar hacer continuos cambios en tiempo de desarrollo que ralenticen el avance del proyecto.
  • Pagos: El Cliente debe realizar un pago del 15% del monto previamente acordado al iniciar el proyecto, esto como muestra del compromiso e interés con el desarrollo acordado, luego un 50% del monto restante una vez el proveedor muestre un avance equivalente aproximado del 50% del proyecto, una vez finalice el desarrollo y sea aprobado por el cliente, este debe hacer el resto del pago para que el proveedor envíe el enlace de descarga de los archivos correspondientes al proyecto.
  • Avances: el proveedor estará en contacto con el cliente para que el mismo este al tanto del avance del proyecto.
  • Cambios: una vez el proyecto haya sido entregado el cliente cuenta con 3 cambios sencillos totalmente libres de costo.
  • Mantenimiento: El cliente puede llegar a un acuerdo de mantenimiento con el proveedor por un costo fijo mensual el cual esta sujeto a cambios según las exigencias progresivas del cliente.

Propiedad Intelectual

  • Derechos de Autor: El cliente puede usar la aplicación como mejor le convenga siempre y cuando haga referencia al autor.
  • Limitación de Responsabilidad: El Proveedor no será responsable por daños indirectos, incidentales o consecuentes derivados del mal uso o la incapacidad de usar apropiadamente el producto final.

Confidencialidad

Ambas partes se comprometen a mantener la confidencialidad de toda la información intercambiada durante la prestación del servicio.

Resolución de Disputas

Cualquier disputa que surja de este acuerdo se resolverá en primer lugar a través de negociaciones amistosas, si un desacuerdo escala y se torna hostil cualquiera de las partes esta en todo el derecho a finalizar la relación de trabajo.

Política de cancelación y reembolso

Una vez realizado el primer pago del 15% el cliente tendrá máximo 24hrs para cancelar y solicitar un reembolso del mismo, una vez transcurrido este tiempo e iniciado el proceso de desarrollo NO habrá devolución del abono realizado, el tiempo del proveedor también tiene un costo.

Aclaración de conceptos técnicos

  • Tecnologías aplicables para el servicio: HTML, CSS, JavaScript, Typescript, Bootstrap, React, NodeJS, PHP, Laravel, Wordpress, MySQL, SQLite.
  • Dominios y Hosting: Esto corre por cuenta del cliente, el mismo ya debe tener establecida y configurada la infraestructura para alojar el desarrollo.
