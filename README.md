* A traves de este proyecto quiero mostrar mis habilidades en el desarrollo front-end
* Es un proyecto que pretendo desarrollar con el tiempo, mejorando y agregando nuevos demos

* Posee proyectos demo que hacen uso de tecnologias como: react, angular, vue, datatables, indexedDB, etc.
